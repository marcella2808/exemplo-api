Bem-vindo ao Sistema de Gerenciamento de Clientes

Este sistema permite gerenciar informações de clientes de forma fácil e prática. Use a barra de navegação para acessar as páginas de cadastro ou visualizar a lista de clientes.

Funcionalidades

  • Cadastro de novos clientes
  • Visualização e edição de informações de clientes
  • Operações CRUD integradas com o MongoDB
